B팀 - 웹 키오스크
박용민, 신도준, 이상윤, 유창모

1. 관리자
	a) 상품등록
	b) 상품옵션 설정 - 수량, 사이드, 기타 등등
	c) 이벤트상품 - 판매기간

2. 판매자 포스기
	a) 과거 주문순으로 주문내역 출력
	b) 주문 확인 버튼
	c) 주문 취소 (환불) 버튼
	d) 수령 완료 버튼

3. 판매자 현황판
	a) 상품 수령 번호 출력
	b) 대기 주문 번호 출력

4. 소비자 일반
	a) 대기열 내림차순 출력
	b) 상품 준비완료시 알림 기능
	c) 상품 목록 버튼
	d) 카드 결제, 포인트, 쿠폰
	e) 주문 취소
	f) 영수증
	g) 장바구니

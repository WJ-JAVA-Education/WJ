 -- 여행 상품 견적 프로그램 --

1. 회원가입
 - 가입 후 게시판 및 견적 내용 확인 가능

2. 로그인
 - 정식 로그인
 - 로그인 없이 견적 요청
 - 로그인한 사용자와 로그인안한 사용자
접근 달리하기

3. 게시판
 - 댓글기능
 - 비밀글 기능
 - 조회수 카운팅

4. 견적요청
 - 간편 견적 요청
 - 상품 클릭으로 견적 바로 산출
(간편견적으로, 가격 변동성 있음 고지)

5. 메일 클릭으로 인한 견적 및 계약서 확정.

6. 상품 검색

관리자 페이지

1. 관리자 회원가입 

2. 상품 관리
- 상품 입력 및 수정.
- 견적 양식 수정

3. 메일 링크
 - 메일로 견적 발송
 - 견적요청 사용자에 광고 페이지 발송

4. 상품 나열 및 상품 검색

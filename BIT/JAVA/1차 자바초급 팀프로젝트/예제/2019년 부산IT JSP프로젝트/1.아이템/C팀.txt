C팀 : 전민철, 이재필, 조기식, 곽훈열, 오재욱
1. 숙박 목록 프로그램(부킹닷컴, 아고다)
- 특정 지역(예:부산) 한정해서 숙박시설들을 등록 
- 여러 필터들(가격, 위치, 방 타입, 편의시설 등)의 체크박스들을 놔두고 
   해당 필터에 부합하는 숙박 시설들을 보여주는 프로그램

2. 영화,연극 크라우드 펀딩사이트(텀블벅,와디즈)
영화와 연극을 한정한 크라우드 펀팅 사이트 
 - 크라우드 펀딩금액을 설정해 참여하고 금액에 차등해서 리워드를 고를 수 있는 방식 
 - 영화의 시놉시스와 감독, 제작사 등의 간단한 정보를 제공하고 펀딩을 할 수 있게함 
 - 펀딩을 신청하는 사람(제작자)이 금액과 리워드를 설정할 수 있음 (최소 2가지 이상의 선택지) 
 - 펀딩 참여자는 미리 설정되어있는 선택지중에서 선택하여 펀딩참여 
 - 목표금액이 100% 달성되어야만 진행 가능 미달되면 프로젝트 무산
 - 펀딩 완료일자까지 펀딩되면 미리 정한 날짜에 한번에 결제 진행 (결제정보는 펀딩신청 당시에 입력) 
 - 내정보에서 확인 가능한 것 (메세지, 내 후원현황, 내 프로젝트, 프로필, 결제정보설정)
- 리워드 발송도 미리 정한 날짜에 순차적 배송 
- 펀딩완료, 실패, 결제완료, 배송출발, 배송도착시 
 홈페이지 내의 메세지를 통해 알려줌
- 메세지는 알림메세지와 유저간 메세지 (제작자,참여자 가리지않음) 가능 

3. 영화 리뷰 사이트
- 영화 리뷰를 작성하는 게시판을 만들고 그 게시판에 글을 작성하거나 
   게시글에 추천 기능을 넣어 받은 추천 수만큼 포인트를 지급
- 포인트를 이용해 기프티콘으로 교환가능하게 하는 등의 
   혜택을 주어 게시판 활동에 대한 동기를 부여
- 영화리뷰 외에도 영화에 관련된 배우들의 인터뷰나 기사거리 
  혹은 각종 시사회의 정보를 작성하는 게시판을 만들어 다양한 정보를 얻을 수 있음

4. 중고차 판매 사이트(sk엔카)
- 국산차와 수입차를 차종, 제조사와 모델, 지역으로 나누어 볼 수 있음
- 동일 중고차를 주행거리, 연식, 가격에 따라 비교 할 수 있음
- 구입 하려는 차량은 등록 되어있는 차량을 찜을 할 수 있으며, 찜한 차량을 목록으로 볼
   수 있으며, 판매 될 경우 찜 목록에서 사라짐
- 판매자는 이름, 휴대폰 번호를 같이 등록 
   구입 하려는 사람은 판매자의 정보를 알 수 있음
- 판매 하려는 차량은 직접 등록(가격, 설명, 연식, 장단점 등)
- 차량에 대한 리뷰를 달 수 있음. 리뷰의 평점이 좋은 차들을 따로 목록로 볼 수 있음

5. 커피 리뷰 시스템(식후 커피 한잔)
실제로 커피 먹는 사람들이 부쩍 늘고 있는 추세입니다
하지만 막상 커피점에 가면 처음 들어서면 빈번히 본인이 먹는 커피만 찾지
새로운 커피에 대한 도전을 하시는 분들이 보통 없기 마련입니다
일일이 검색하며 찾아봐야만 알 수 있는 부분이 있어
하나 만들어봐도 괜찮을 거 같아서 올립니다
- 특정 브랜드 카페 메뉴마다 커피 전문가?가 리뷰를 써놓아 목록으로 정리
- 유명 체인점부터 시작하여 편의점 커피까지 페이지 정리
(유저마다 애용하는 커피점이 있기 마련이기에 각 넘어가는 페이지 생성)
- 웹을 사용하는 유저들도 리뷰 참여 가능
- 리뷰 평가가 높거나 자주 글쓰는 유저에겐 특정 등급 승격(지식인 같은)
- 주,월마다 인기 커피를 투표로 인해 선정해 메인 페이지에 표시
- 불호 음료 및 인기 음료를 그래프로 정리
- 오늘의 추천 커피 및 음료는?
(커피에 고민 많은 유저들에게 가볍게 추천하기도 좋은 웹 메인에 표시)
- 나이,성별에 따른 커피 선호도 정리
  (나이대에 따른 정리가 신뢰도 높지 않을까 합니다)
-실시간 검색어
  (만들 수 있다면 이런 것도 좋지 않을까 합니다.)
